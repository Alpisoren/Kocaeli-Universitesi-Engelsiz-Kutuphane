Kocaeli Engelsiz Kütüphane web sayfasına hoş geldiniz.

Sesli kitap hizmetimize kütüphanemiz ana sayfasından http://library.kocaeli.edu.tr/yordambt/yordam.php adresinden ulaşabilirsiniz. Web sayfamızı daha kolay bir şekilde kısa yol tuşları ile kullanabilirsiniz. İstediğiniz sayfaya gitmek için ilgili tuşlara basmanız yeterlidir.

Alt +a---tüm sesi aç / tüm sesi kapat
Alt + 0 = İçerik okuma
Alt + (-) = Yazı küçültme
Alt + k = İçerik durdurma
Alt + (*) = Renk Değiştirme
Alt + b = Bilgilendirme
Alt + V = Varsayılana dön
Alt + (+) = Yazı büyütme

Web sayfamızın sağ köşesinde bulunan "Yazı Boyutu" tuşunu kullanarak ekrandaki yazıların boyutunu ayarlayabilirsiniz Web sayfamızın sağ köşesinde bulunan "Renk Değiştirme" tuşunu kullanarak ekrandaki içeriğin rengini ayarlayabilirsiniz Web sayfamızın sağ köşesinde bulunan "Varsayılan" tuşunu kullanarak içeriğimizi varsayılan haline ayarlayabilirsiniz
